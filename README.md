## How to Run Argus
1. Install requirements.txt
1. Run Backend Services
   1. RunMaster.py
   1. RunDetect.py
   1. RunTracker.py
   1. RunCrash.py
1. Run Client Services
   1. RunGui.py
   1. RunCamera.py
1. Select video from videos folder
1. From RunCamera.py hit Process
